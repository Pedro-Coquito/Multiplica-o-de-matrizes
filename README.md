Esse codigo foi criado para o Curso de Engenharia de Computação para atender os requerimentos da materia de estrutura de dados.

A multiplicação de matrizes de n linhas para n colunas foi escolhida para submeter a este trabalho por conta de sua principal função ter custo polinomial ou cubica n^3, ultilizando-se de 3 loops for aninhados 
para a realização da multiplicação.
